Создать плагин для поиска фильмов на https://www.omdbapi.com/ . 
В админской части плагин должен иметь страницу настроек 
(АПИ-ключ, лимит фильмов в выдаче по умолчанию, тип фильмов по умолчанию: сериал, фильм). 

Для фронтенда плагин должен обрабатывать шорткод. 
Шорткод выводит форму для поиска по фильмам и результаты поиска. 
В форме по умолчанию выставлен тип фильма и лимит из настроек плагина 
в админской части.
